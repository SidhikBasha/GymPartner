# 💪 GYM PARTNER — Daily To Do List

A motivational daily gym workout checklist built with pure HTML and CSS. No JavaScript.

---

## 📌 Features

- Clean dark industrial design with fiery accent colors
- Interactive checklist with smooth check animations and line-through effect
- Progress bar styled like a barbell
- Well-organized sections: Warm Up, Strength, Cardio, Core & Recovery
- Realistic workout details (sets, reps, rest times, tags)
- Animated elements — pulse glow, shimmer hover effects, staggered entrance
- Noise texture overlay for authentic gym feel
- Fully responsive design
- Motivational marquee footer

## 🛠 Tech Stack

- **HTML5**
- **CSS3** (Custom Properties, Flexbox, Grid, Keyframe Animations)

## 🎯 Purpose

To practice advanced CSS animations, modern layout techniques, and create a beautiful, functional single-page fitness tracker.
